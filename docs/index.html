공부 중

기획 / PM

CMS·시스템 설계 경험을 기반으로 기획과 프로젝트 관리 영역까지 시야를 확장해가고 있습니다.

강점

문제 원인을 끝까지 추적하는 디버깅 능력 — 원인 불명 버그도 로그·쿼리·코드 역추적으로 해결
백엔드·DB·운영을 함께 고려하는 실무형 시각 — 개발 단계부터 운영 이슈를 미리 설계에 반영
요구사항을 기능 단위로 구조화하는 능력 — 모호한 고객 요구사항을 구체적 기능 명세로 전환
팀장 역할 이후 일정 지연 없이 프로젝트를 운영한 실행력과 마감 준수 능력
사용자/운영자 관점까지 반영하는 개발 방식 — UX와 관리 편의성을 동시에 고려한 설계

성장 중인 부분

책임감이 강해 혼자 많은 범위를 챙기려는 경향 — 최근에는 우선순위와 역할 분담을 더 명확히 하며 보완 중
세부 완성도·커뮤니케이션 품질까지 신경 쓰다 보니 에너지 소모가 큰 편 — 효율적 협업 방식으로 개선 중
서버 인프라·배포(CI/CD) 경험 부족 — 클라우드·DevOps 영역 지속 학습 중

앞으로 나아갈 방향

🤖 AI를 활용한 개발로의 전환 — LLM 기반 서비스, AI 보조 개발 도구 연구
🔎 QA 전문성 강화 — 테스트 자동화, 품질 관리 체계 구축 역량 개발
📄 서비스 기획 능력 향상 — 사용자 중심 설계, 요구사항 분석 심화 학습
